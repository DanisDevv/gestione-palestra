# Gestione Prenotazioni Palestra

**Descrizione:**  
Sistema web per gestire le prenotazioni di una palestra fittizia, “FitLife Gym”. Permette ai clienti di prenotare corsi e agli admin di gestire le prenotazioni in modo semplice e veloce.

**Tecnologie utilizzate:**  
- Java (Spring Boot)  
- H2 / MySQL per il database  
- HTML, CSS, JavaScript per il frontend  

**Funzionalità principali:**  
- Visualizzazione orari e corsi disponibili  
- Prenotazione corsi da parte dei clienti  
- Login admin per gestione prenotazioni  
- CRUD di prenotazioni (Create, Read, Update, Delete)
